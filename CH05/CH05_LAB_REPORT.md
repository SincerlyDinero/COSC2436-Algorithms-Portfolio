## Reflection Questions

1. **What is a hash function?**  
A hash function converts a key into an index location.

2. **What is a collision?**  
A collision occurs when multiple keys map to the same index.

3. **Why are hash tables efficient?**  
Hash tables allow fast lookup operations in constant average time.
