# COSC 2436 - Algorithms Portfolio

A semester-long collection of 12 chapter labs covering data structures and algorithms in Python. Each chapter folder contains working program files and a detailed lab report explaining the algorithm, test results, and reflections.

## Author
Nathan Higgins - COSC 2436, Spring 2026

## Description

This repository demonstrates my understanding of fundamental data structures and algorithms including searching, sorting, recursion, graph traversal, trees, shortest path algorithms, greedy methods, dynamic programming, and introductory machine learning concepts.

Throughout the semester, I developed programs implementing these algorithms while analyzing their efficiency and practical applications.

